# Gestão de utilizadores

Acessível apenas ao perfil **Administrador** em **Utilizadores** no menu.

## Papéis (roles)

| Papel | Acesso |
|---|---|
| **Administrador** | Gerir utilizadores; acesso total |
| **Profissional de saúde** | Doentes, fichas, modelos, categorias, estatísticas |
| **Cuidador** | Registar sintomas, consultar fichas do seu doente |

## Criar utilizador

1. **+ Novo utilizador**
2. Preencha nome, email, papel e palavra-passe inicial
3. O utilizador deve alterar a palavra-passe no primeiro acesso

!!! tip "Criar cuidadores"
    Os profissionais de saúde também podem criar cuidadores directamente em **Doentes → Novo doente → Criar novo cuidador**, sem passar pela área de administração.

## Desactivar utilizador

Clique em **Desactivar** na lista de utilizadores. O utilizador fica impossibilitado de entrar na aplicação. Os dados associados são preservados.

!!! warning "Sessões activas"
    A desactivação é verificada em cada pedido. Uma sessão activa é invalidada na próxima página carregada.

## Repor palavra-passe

Clique em **Editar** e defina uma nova palavra-passe. Communique ao utilizador por canal seguro.

## Não é possível eliminar utilizadores

Utilizadores não podem ser eliminados para preservar a integridade dos registos históricos (registos de sintomas, mensagens). Use a desactivação.
