# Primeiros passos — Cuidador

## Aceder à aplicação

1. Abra o browser e aceda ao endereço fornecido pela equipa clínica
2. Introduza o email e palavra-passe que lhe foram atribuídos
3. Na página inicial verá a lista dos doentes sob o seu cuidado

!!! tip "Dificuldade a entrar?"
    Contacte o profissional de saúde responsável para repor a palavra-passe. Não é possível recuperar a palavra-passe de forma autónoma.

---

## Alterar a palavra-passe

Após o primeiro acesso, é recomendável alterar a palavra-passe:

1. Clique no seu nome (canto superior direito)
2. Introduza a palavra-passe actual e a nova palavra-passe (mínimo 8 caracteres)
3. Confirme e guarde

---

## Ecrã principal

Após o login, vê a lista de doentes atribuídos. Cada doente tem:

- **Nome/código** — identificador atribuído pela equipa
- **Diagnóstico principal**

Clique no nome do doente para abrir a sua ficha.

---

## Navegação rápida

| Botão/Link | Função |
|---|---|
| **+ Registar sintomas** | Fazer o registo diário |
| **Ver evolução** | Gráficos dos registos anteriores |
| **Informação** | Fichas de informação clínica |
| **Referências** | Base científica das fichas |
| **Ajuda** | Manual de utilização |
