# CUIDAR

**Sistema de apoio ao cuidador informal em cuidados paliativos**

Desenvolvido pela Faculdade de Medicina da Universidade do Porto (FMUP), o CUIDAR apoia o cuidador informal no acompanhamento diário de pessoas em situação paliativa, facilitando a comunicação com a equipa clínica e o acesso a informação clinicamente validada.

---

## Para quem é este sistema?

| Perfil | O que pode fazer |
|---|---|
| **Cuidador** | Registar sintomas diários, consultar fichas de informação, enviar mensagens à equipa clínica |
| **Profissional de saúde** | Gerir doentes, criar fichas de informação, monitorizar alertas, consultar estatísticas |
| **Administrador** | Gerir utilizadores e acessos |
