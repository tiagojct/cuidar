# Estatísticas

Aceda em **Estatísticas** no menu de navegação.

## Painel de indicadores

| Indicador | Descrição |
|---|---|
| Total de doentes activos | Doentes não arquivados |
| Total de registos | Entradas de sintomas no sistema |
| Dias de alerta (30 dias) | Registos com bem-estar ≤ 3 ou sintoma ≥ 7 nos últimos 30 dias |
| Bem-estar médio (30 dias) | Média do bem-estar geral de todos os doentes |
| Fichas mais consultadas | Top fichas de informação por visualizações |

## Exportar dados

- **Exportar todos (CSV)** — exporta todos os doentes activos (lista de doentes, página principal)
- **Exportar doente (CSV)** — exporta todos os registos de um doente específico

## Relatório por doente

Na ficha de cada doente (visão clínico ou cuidador), clique em **Ver evolução → Imprimir relatório** para um relatório completo imprimível.
